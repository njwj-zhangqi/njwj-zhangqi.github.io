Linux命令

系统类

  • lsb_release -a 查看操作系统版本。若无法使用，先um install lsb

  • uname -a 查看操作系统信息

  • free -m查看内存

  • df -lh查看磁盘

  • top查看CPU和内存

  • du -sh查看当前文件总大小

  • ls -lh 查看文件详细信息

排查

  • ps -ef | grep java ，查询java进程，前面的数字是PID
