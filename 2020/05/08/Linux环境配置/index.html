Linux环境配置

ifconfig 无法使用

Host无法识别

问题：Could not resolve host: mirrorlist.centos.org; Unknown error

wget 找不到

wget无法解析主机地址

  1. ping 8.8.8.8 确定网络没问题
  2. 设置了域名服务器
  3. 确保网关已设置

yum install 报错

问题：There are no enabled repos.

解决：使用wget下载新的repo

VMware配置静态IP

  1. 编辑 ==> 虚拟网络编辑器
  2. 获取管理员模式 ==>选中NAT模式 ==> 设置底部的子网IP 192.168.170.0 ==> 点击NAT设置 ==> 填写网关IP192.168.170.2
  3. 配置虚拟机中的网卡文件vi /etc/sysconfig/network-scripts/ifcfg-ens33
  1. 重启网络服务
