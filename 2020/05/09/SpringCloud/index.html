Docker

安装Dock

Docker搭建WordPress

  1. docker命令
  • docker image inspect mysql:latest | grep -i version 查看latest标签的具体版本
  • docker ps -a 查看所有的容器
  • Tdocker rm 容器id 删除指定id的容器
  • docker rmi 镜像名 删除镜像(先删除container)
  • docker -v 查看docker版本
  • /var/lib/docker docker默认安装木来说
  • docker inspect 容器名 查看具体信息
  • docker kill $(docker ps -a -q) 杀死所有正在运行的容器
  • docker rm $(docker ps -a -q) 删除所有已经停止的容器
  • docker rmi $(docker images -q) 删除所有镜像
  1. Job for docker.service failed because the control process exited with error code

一般是/etc/docker/daemon.json文件写错

  1. 配置docker镜像

vi /etc/docker/daemon.json

升级docker到新版本

  • 查看docker安装过的包 yum list installed | grep docker

  • 移除安装包 yum remove 名称

  • 安装新版本的包 yum install docker-ce docker-ce-cli containerd.io

  1. Error establishing a database connection

    docker在创建mysql容器时指定了创建数据库的密码,wordpress配置文件中数据库的连接密码默认为空，所以出现上面的错误。

    解决:

彻底删除Docker
