Redis命令

Redis操作都是原子的，所以不用考虑并发访问。如果key不存在时，会自动创建key，value为空，会删除key

String

命令 示例 解释
set set user:100 “Hello” 添加/更新user，可以用:来拼接key，user:100是一个key
get get user 获取user的值
exists exists user 查看user是否存在
del del user 删除user
incr incr user user对应的Value自增1
incrby incrby user 100 user对应的Value指定增长100
decr decr user user对应的Value自减1
decrby decrby user 100 user对应的Value指定减少100
expire expire user 10 指定user过期时间为10s，单位：second
ttl ttl user 返回值：剩余过期时间。-2：key不存在，-1：key永不过期，单位：秒
persist persist user 指定user永不过期

复合命令：SET key 123 EX 6 => 设置key的值为123，过期时间为6s

List

命令 示例 解释
rpush rpush user “1” “2” “3” 在尾部插入值，value是 1、2、3
lpush lpush user “4” 在头部插入值
llen llen user 查看user长度
lrange lrange user 1 2 查看user索引从1~2位置的元素
lpop lpop user 移除第一个元素，并返回移除的元素
rpop rpop user 移除最后一个元素，并返回移除的元素

SET

命令 示例 解释
sadd SADD key value 如果添加的value存在，返回0，不存在返回1
srem srem key 移除key
sismember sismember key value key中是否存在value
smembers smembers key 查看key的所有值
sunion sunion key1 key2 取并集
spop spop key 随机返回元素，并移除
srandmember srandmember key 随机返回key中的一个元素

SORTED SET

与SET很类似，只是进行了排序

命令 示例 解释
zadd zadd key score value 添加元素
zrange zrange key start stop 和lrange类似，返回指定key范围的元素

ZADD key score value，每个值都有一个score，按score来排序

HASH SET

命令 示例 解释
hset HSET key name “John”
HSET key age “12”
HSET key password “afdsa”		 指定key，然后后面是属性和值
hgetall HGETALL key 获取key中的所有Map
hget HGET key subkey 获取key中指定属性的值
hincrby HINCRBY key visit 1 指定key中属性的值增加多少
hdel HDEL key visit 删除key中某个属性
